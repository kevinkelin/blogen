Application of for range in go language

Publish: 2019-08-11 Classify at: Go
Words:448 Read:≈ 3min

for range is used in go language to traverse an iterable object, it can be applied to arrays, slices, dictionaries, etc. It has the following usages

Traverse arrays and slices

package main

import "fmt"

func main() {
	a := [...]string{"a", "b", "c", "d"}
	for i := range a {
		fmt.Println("Array item", i, "is", a[i])
	}
}

When initializing an array, use [...]string to initialize a string array, the length is automatically calculated, if written as []string, it is to initialize a slice.

for i := range a Here is by traversing the subscript of the array, the subscript of i is incremented by 1 each time, and then the value of the array is obtained by array[i], or it can be obtained directly in the for range structure array element

package main

import "fmt"

func main() {
	var fruits = map[string]int {
	"apple": 2,
	"banana": 5,
	"orange": 8,
	}

	for f,s := range fruits{
		fmt.Println("fruits:",f,"score:",s)
	}
	for f := range fruits{
		fmt.Println("fruits:",f,"score:",fruits[f])
	}
}

traverse the channel

You can use for range to traverse the channel. When the channel is not closed, it will block and wait. When the channel is closed, it will jump out of the loop.

package main

import "fmt"

func main() {
	ch := make(chan int,6)
	go setData(ch)
	for ele := range ch{
		fmt.Println("get data " ,ele)
	}
}

func setData(ch chan int) {
	for v:=0;v<8;v++{
		ch <- v
	}
	close(ch)
}

If in the above code, the setData function does not close(ch) to close the channel, panic deadlock will occur

The above for range traverses the channel, if you use a for loop to write it as follows

package main

import "fmt"

func main() {
	ch := make(chan int,6)
	go setData(ch)
	for{
		v,ok := <- ch
		if !ok{
			break
		}
	fmt.Println("get data ",v)
	}
}

func setData(ch chan int) {
	for v:=0;v<8;v++{
		ch <- v
	}
	close(ch)
}
