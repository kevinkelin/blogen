Summary of the use of file paths in the go language

Publish: 2019-08-25 Classify at: Go
Words:362 Read:≈ 2min

It is very common to manipulate files in programming, such as reading or writing files, but before that, the file path will be constructed. There is a corresponding os.path library in python, and the operation of file paths is very simple, such as

  • os.path.basename to get the filename
  • os.path.dirname to get the file directory name
  • os.path.join concatenate file path
  • os.path.splitext split file, get filename and extension
  • os.path.adspath Get the absolute path of the file
  • os.path.exists to determine whether a file or folder exists

The following attempts to implement the above methods in the go language

Manipulating file paths in the go language mainly uses the filepath package, which is relatively simple to use and has been written in the comments.

package main

import (
	"fmt"
	"os"
	"path/filepath"
)

func main() {
	pwd,_ := os.Getwd() // Get the current directory, equivalent to os.getcwd() in python
	fmt.Println("The current operation path is:",pwd)
	// file path concatenation
	f1 := filepath.Join(pwd,"test","1.txt")
	fmt.Println("The path of the file is: ",f1)
	// directory name of the file
	fmt.Println("File directory name:",filepath.Dir(f1))
	// filename of the file
	fmt.Println("File name of the file:",filepath.Base(f1))
	// absolute path to the file
	adspath,_ := filepath.Abs("evn/3.txt")
	fmt.Println("The absolute path of the file is: ",adspath)
	// split path
	dirname,filename := filepath.Split(f1)
	fmt.Println("Directory name:",dirname,"File name",filename)
	//Extension related
	fmt.Println("The extension of f1 is: ",filepath.Ext(f1))

	//The file status returned by the os.Stat() function, if there is an error, it is judged whether the file or folder exists according to the error status
	fileinfo,err :=os.Stat(f1)
	if err!=nil{
		fmt.Println(err.Error())
	if os.IsNotExist(err){
		fmt.Println("file:",f1,"not exist!")
	}
	}else{
		// Determine if the path is a folder
		fmt.Println(fileinfo.IsDir())
		fmt.Println(!fileinfo.IsDir())
		fmt.Println(fileinfo.Name())
	}
}

The result of running is:

The absolute path of the file is: F:\GO\src\gobookcode\evn\3.txt
The directory name is: F:\GO\src\gobookcode\test\ The file name is 1.txt
The extension of f1 is: .txt
