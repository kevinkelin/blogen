def callbackfunc(task):

loop.run_until_complete is a blocking method. Only when the coroutine in it finishes running, the method ends, and the following code is run.

In fact, you can also not call the loop.run_until_complete method. After creating a task, the coroutine function is already running, but when the event loop is ready to run, the task status at this time is pending, if not If the event loop is called, the coroutine function will not be run. Since the main thread runs out, the child thread will be destroyed, as the code is written as follows:

So if you want to make the coroutine function execute, you need to call the event loop to execute the task. The above loop.run_until_complete is to make the loop start running. In fact, you can also use loop.run_forever(), this function is just like its Same name, will keep running. Only when the event loop is running, the coroutine registered with the loop will be executed, but if loop.run_forever() is used, it will block here. The event loop also has a stop method, which can end the loop, We can add a callback method to the task object. When the execution of the coroutine ends, call the stop method of the event loop to end the entire loop

#coding:gbk
import asyncio
import time,sys

async def asyncfunc1():
    print("Suspending func1")
    await asyncio.sleep(1)
    print("func func1 ", threading.current_thread())
    print('Resuming func1')
    return "func1"
    
# define a callback function
def callbackfunc(task):
    print("task is finished, its result is: ", task.result())
    loop.stop()
    
    
if __name__=="__main__":
    print("In main thread ",threading.current_thread())
    loop = asyncio.get_event_loop()
    task = loop.create_task(asyncfunc1())
    task.add_done_callback(callbackfunc)
    loop.run_forever()

In addition to using the loop.run_until_complete method, you can also use the asyncio.ensure_future() method to run the coroutine, change task = loop.create_task(asyncfunc1()) in the above code to task = asyncio.ensure_future (asyncfunc1())will get the same result. Its parameter is a coroutine object or futures. You can also pass a task object, because task is a subclass of futures. When a coroutine object is passed in, a task is returned. Object, when a futures is passed in, the futures object is returned directly, that is, after callingasyncio.ensure_future()`, it will return a task object, you can add a callback method to it, and you can call task.result () method to get the result (note that if the result method is called before the task execution ends, an exception will be thrown).

#coding:gbk
import asyncio

# define two async functions
async def asyncfunc1():
    print("Suspending func1")
    await asyncio.sleep(1)
    print("func func1 ", threading.current_thread())
    print('Resuming func1')
    return "func1"
    
async def asyncfunc2():
    print("Suspending func2")
    await asyncio.sleep(1)
    print("func func2 ", threading.current_thread())
    print('Resuming func2')
    return "func2"# define a callback function
def callbackfunc(task):
    print("task is finished, its result is: ", task.result())
    
    
if __name__=="__main__":
    print("In main thread ",threading.current_thread())
    loop = asyncio.get_event_loop()
    task1 = loop.create_task(asyncfunc1())
    task1.add_done_callback(callbackfunc)
    task2 = loop.create_task(asyncfunc2())
    task2.add_done_callback(callbackfunc)
    loop.run_forever()

At this point, because the loop calls the run_forever method, and there is no method to call the stop method, the program will always be stuck.

In this way, multiple coroutines can be run, but such processing is cumbersome and inconvenient for the recovery of results.

asyncio has a gather method that can pass in multiple task objects. When calling await asyncio.gather(*), it will return all the results

Since await can only be written in async def functions, a new function needs to be created here

    task1.add_done_callback(callbackfunc)
    task2 = loop.create_task(asyncfunc2())
    task2.add_done_callback(callbackfunc)
    result = await asyncio.gather(task1,task2)
    print(result)
